## 🏁 Para rodar o projeto:

Clone este repositório em sua máquina:

```bash
$ git clone https://github.com/araujooleticiaa/template-nodejs-typescript
```

cd `template-nodejs-typescript` e rode:

```bash
yarn
```

para iniciar:

```bash
yarn dev:server
```
